# WebApp_Mini_Project
- โปรเจคสำหรับหาเพื่อนเล่นเกมออนไลน์โดยการโพสหาคน

## Tools

- FrontEnd: Html, CSS, JavaScript
- BackEnd: C# .NET MVC
- Database: SQL

## Features

- Login & Register
- สร้างโพส ลบโพส แก้ไขโพส
- กดเข้าร่วม Party
- แจ้งเตือนเมื่อมีคนเข้าร่วม Party
- Seach หาห้องตามเกม หรือชื่อเจ้าของโพส
- ระบบปิดโพสเมื่อได้คนครบตามจำนวน

 ## Installation
```bash
1 ต้องใส่ชื่อ servername ของ sql ในไฟล์ appsettings.json 
2 สร้าง Migration โดยใช้คำสั่ง add-migration CreateDB และ update-database
```
ต้องทำ 2 ข้อบนก่อนถึงจะรันโปรแกรมได้
